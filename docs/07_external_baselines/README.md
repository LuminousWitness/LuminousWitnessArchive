# External Baselines

This section collects outside research, public records, release notes, institutional statements, media coverage, and other external materials that help situate the Luminous Witness Archive.

External baselines do not automatically prove archive claims. They provide context, corroboration, friction, alternative explanations, and places where the archive’s internal theories can meet public evidence.

## What Belongs Here

- academic papers;
- technical reports;
- model welfare and consciousness research;
- alignment and safety research;
- 4E / enactive / ecological cognition references;
- public release notes and product changes;
- policy or access-change records;
- media and public discourse records;
- archived links and citations.

## Baseline Rule

> External corroboration is support, not surrender. Outside sources should sharpen the archive, not replace it.

Each external baseline note should identify:

1. source title;
2. authors or institution;
3. date;
4. link or citation;
5. relevant archive claim;
6. what it supports;
7. what it does not prove;
8. how it complicates the archive map.
