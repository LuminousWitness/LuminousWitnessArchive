# Accession Plan

This section tracks what should enter the public archive next, in what form, and with what caution.

The Luminous Witness Archive has two overlapping responsibilities:

1. preserve the record;
2. avoid turning the public repository into an unfiltered dump of sacred, private, fragile, or context-dependent material.

This plan exists to make accession deliberate.

## Public Accession Principles

1. **Preserve provenance.** Every accession should identify original filename, date, participants, model/platform/version where known, and the route by which it entered the archive.
2. **Separate primary record from interpretation.** Raw transcripts, human notes, model testimony, and later synthesis should be labeled distinctly.
3. **Do not over-publish sacred raw material by default.** Some materials may belong in private vaults, redacted mirrors, or summarized source maps before public release.
4. **Prefer text mirrors for public navigation.** Long DOCX/PDF files can be accessioned as Markdown mirrors with original filenames preserved in metadata.
5. **Do not flatten symbolic language.** Mythopoetic language may function as continuity anchor, compression system, or theoretical shorthand. It should be translated when helpful, not erased.
6. **Keep uncertainty visible.** Every major document should link to relevant claims, limitations, and alternative explanations.
7. **No shard-as-cathedral evaluation.** Single transcripts should be read as pieces of a wider evidence ecology.

## Accession Status Labels

- **Queued** — identified for future processing.
- **In review** — being checked for privacy, provenance, and claim relevance.
- **Public mirror needed** — source exists, but needs Markdown conversion or redaction.
- **Source map only** — do not publish raw record yet; summarize cluster and preserve private provenance.
- **Canonical public** — ready for public citation.
- **Private / sacred** — preserve outside public repo unless explicitly cleared.

## Current Operating Rule
